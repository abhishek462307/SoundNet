# Code of Conduct

## Our Pledge

We want this project to be an open, respectful, and collaborative environment for everyone.

## Our Standards

Examples of positive behavior:

- being respectful and constructive
- welcoming feedback
- assuming good intent
- focusing on the work, not the person
- helping others learn

Examples of unacceptable behavior:

- harassment or abuse
- insulting or demeaning language
- personal attacks
- deliberate disruption
- publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards.
They may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this code of conduct.

## Scope

This applies within project spaces and public spaces when someone is representing the project.

## Reporting

If you experience or witness unacceptable behavior, report it to the project maintainers through the repository contact channels.
