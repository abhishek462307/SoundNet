# Sound Net Docs

Welcome to the Sound Net documentation.

**Sound Net** stands for:

**Service Orchestration Unified Network for Discovery, Navigation, Execution, and Transport**

Sound Net is open source infrastructure for:

- tool discovery
- MCP server aggregation
- tool execution routing
- agent discovery
- agent messaging
- operator analytics
