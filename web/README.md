# Sound Net Operator Console

Vite + React dashboard for operating a running `Sound Net` backend.

## Local development

```bash
npm install
npm run dev
```

Default Vite URL:
- `http://127.0.0.1:5173`

Configure the dashboard in the sidebar with:
- Sound Net base URL
- API key
- admin key
- tenant id

## Current UI scope

- overview dashboard
- health and readiness status
- analytics summary
- top tools and policy analytics
- capabilities browser
- MCP server browser
- execution preview

## Next planned screens
