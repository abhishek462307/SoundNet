# Contributing

Thanks for your interest in contributing to Agent Network.

## How to contribute

You can help by:

- reporting bugs
- suggesting features
- improving documentation
- submitting bug fixes
- improving tests
- proposing architecture improvements for agent/tool/MCP workflows

## Development setup

```bash
npm install
npm run dev
npm test
```

## Contribution guidelines

- keep changes focused and minimal
- prefer fixing root causes over patching symptoms
- add or update tests when behavior changes
- keep documentation in sync with code changes
- avoid unrelated refactors in the same PR

## Pull requests

Please include:

- what changed
- why it changed
- how it was tested
- any follow-up work still needed

## Issues

When filing issues, include:

- expected behavior
- actual behavior
- reproduction steps
- relevant logs or API responses

## Design direction

This project is evolving around:

- tool discovery
- MCP aggregation
- agent discovery
- agent messaging
- trust and analytics

Contributions aligned with that direction are especially helpful.
